# Roast & Route

A fintech app that hilariously roasts your impulse purchases using AI while routing essential payments smoothly.

## Features
- AI-Powered Transaction Routing
- Three Personalities (Chill, Sassy, Ruthless)
- Simulated UPI Payment Gateway
- Dynamic Camera QR Scanner

## Installation

1. Copy the repository to your local machine.
2. Run `npm install` to install dependencies.
3. **IMPORTANT**: You must create a `.env` file in the root directory to run this project. Add the following line to your `.env` file with your Gemini API key:
   `VITE_GEMINI_API_KEY=your_api_key_here`
4. Run `npm run dev` to start the local development server.

# Gloop: Roast & Route

A fintech app that hilariously roasts your impulse purchases using AI while routing essential payments smoothly.

## Problem Statement
- **The Cashback Dependency:** Current fintech giants (Paytm, GPay) burn millions on cashbacks that buy zero loyalty. Users leave the second the free money stops.
- **The Leaky Bucket:** Cashback is a "leaky bucket" strategy—it buys engagement but destroys financial health by encouraging junk spending.

## Solution: Roast & Route + The ₹10 Flip
- **Roast & Route:** Replace "free money" with gamified friction. Bloop roasts unnecessary scans (₹400 coffees) and challenges users to save.
- **The ₹10 Flip:** Instead of a boring ₹10 cashback, Gloop automatically "flips" that value into Digital Gold or Mutual Funds. We replace "spending addiction" with "growth addiction."
## Features
- AI-Powered Transaction Routing
- Three Personalities (Chill, Sassy, Ruthless)
- Simulated UPI Payment Gateway
- Dynamic Camera QR Scanner

## Installation

1. Copy the repository to your local machine.
2. Run `npm install` to install dependencies.
3. **IMPORTANT**: You must create a `.env` file in the root directory to run this project. Add the following line to your `.env` file with your Gemini API key:
   `VITE_GEMINI_API_KEY=your_api_key_here`
4. Run `npm run dev` to start the local development server.
